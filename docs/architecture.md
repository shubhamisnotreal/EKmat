# EkMat System Architecture

EkMat is a privacy-first e-voting system Pilot that leverages Zero-Knowledge Proofs (ZKPs) to guarantee voter anonymity while ensuring verifiable results. The architecture consists of four main layers:

## 1. Blockchain Layer (Smart Contracts)
- **Role**: The decentralized source of truth for election configuration, vote counting, and proof verification.
- **Components**: 
  - `EkMatVoting.sol`: The core voting contract. It stores election metadata, candidate lists, and vote tallies. It enforces the one-voter-one-vote rule using ZKP nullifiers.
  - `Verifier.sol`: A smart contract automatically generated by SnarkJS/Circom to cryptographically verify the Zero-Knowledge proofs on-chain.
- **Environment**: Developed with Hardhat and deployed locally to Ganache.

## 2. Privacy Layer (Zero-Knowledge Proofs)
- **Role**: Ensures a user is eligible to vote without revealing their identity.
- **Components**:
  - `voterEligibility.circom`: The ZK circuit written in Circom. It takes private inputs (commitment, nullifier, Merkle path) and public inputs (Merkle Root, Election ID) to generate a unique `nullifierHash`.
  - **Tree Structure**: Eligibility is managed via a Merkle Tree of registered voter commitments.

## 3. Backend Layer (Node.js/Express)
- **Role**: Off-chain services provider and relay.
- **Components**:
  - **ZKP Service**: Currently mocks proof generation and builds Merkle Trees for registered voters.
  - **IPFS Integration**: Uploads candidate photos and anonymous vote receipts to IPFS for decentralized storage.
  - **EkSaathi AI**: A multilingual support chatbot integrated with Google Gemini API to assist voters in English, Hindi, and Marathi.

## 4. Application Layer (React + Vite)
- **Role**: The user-facing portal for admins and voters.
- **Components**:
  - **Admin Dashboard**: For creating elections and managing the Merkle Root.
  - **Voter Portal**: Integrates with wallet providers (Ethers.js), handles identity verification mock-ups (DigiLocker), and initiates the voting flow.
